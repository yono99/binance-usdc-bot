## Risiko & mitigasi
- **Migration SQLite**: idempotent ALTER TABLE + DEFAULT; test migrasi di DB copy dulu.
- **State restore backward-compat**: state lama (`balance` tunggal) tetap terbaca, di-migrasi in-memory.
- **Live API rate limit**: set margin isolated + leverage di-cache per simbol (jangan ulang tiap entry).
- **WS per-mode proses ganda**: dokumentasikan satu proses per mode (sudah pattern `forwardtest.py --mode`).
